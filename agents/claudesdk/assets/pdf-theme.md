# Tema visual EDteam para reportes PDF

Aplica este tema en todos los PDFs generados con reportlab.

## Paleta de colores

| Rol               | Hex       |
|-------------------|-----------|
| Fondo de página   | `#FFFFFF` |
| Header            | `#0d1b3e` |
| Título en header  | `#FFFFFF` |
| Títulos sección   | `#3b82f6` |
| Subtítulos        | `#0d1b3e` |
| Texto normal      | `#1e293b` |
| Texto secundario  | `#64748b` |
| Filas alternas    | `#f1f5f9` |
| Bordes de tabla   | `#cbd5e1` |
| Acentos / líneas  | `#f59e0b` |
| Footer            | `#94a3b8` |

## Estilos por elemento

### Página
- Fondo blanco, márgenes de 40pt en todos los lados.

### Header
- Rectángulo `#0d1b3e` de ancho completo y 60pt de alto.
- Título del reporte en `#FFFFFF`, Helvetica-Bold 18pt, centrado.

### Títulos de sección (Heading1)
- Helvetica-Bold 13pt, color `#3b82f6`.
- Línea horizontal de 1pt debajo en `#3b82f6`.

### Subtítulos (Heading2)
- Helvetica-Bold 11pt, color `#0d1b3e`.

### Texto normal
- Helvetica 10pt, color `#1e293b`, interlineado 14pt.

### Texto secundario / metadatos
- Helvetica 9pt, color `#64748b`.

### Tablas
- Encabezado: fondo `#0d1b3e`, texto `#FFFFFF`, Helvetica-Bold 10pt.
- Filas alternas: `#f1f5f9` y `#FFFFFF`.
- Bordes: `#cbd5e1`, grosor 0.5pt.

### Badges / géneros
- Rectángulo redondeado, fondo `#3b82f6`, texto `#FFFFFF`, Helvetica 8pt.

### Separadores
- Líneas horizontales de 1pt en `#f59e0b`.

### Footer
- Línea fina `#cbd5e1` seguida de texto centrado en `#94a3b8`, Helvetica 8pt, con número de página.
